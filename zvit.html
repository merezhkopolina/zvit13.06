Опис проєкту (WEB застосунку):

WEB-застосунок – система бронювання столиків у ресторані. Користувач може обрати дату, час, кількість осіб, ім’я та email. Дані зберігаються в базі даних MongoDB через бекенд на Node.js з використанням Express і Mongoose.

Основні технології:

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • База даних: MongoDB
  • Інші: Vite, Axios, Docker

Висновок:

У рамках курсу я створила повнофункціональний веб-додаток для резервування столиків. Було використано всі ключові технології з фронтенду та бекенду. Сервер коректно обробляє запити, працює з базою даних, а клієнт має зручний UI для користувача.
